# Address Updates for Google Sheet

**Note:** I cannot edit your Google Sheet directly because it requires your Google sign-in. I've updated the local CSV file (`1MHR_All_Teams_Expanded.csv`) with **48 new full addresses** using the rink names from columns E and F.

## How to Update Your Google Sheet

### Option A: Re-import the whole CSV (easiest)
1. Open your Google Sheet: https://docs.google.com/spreadsheets/d/1euwHwQX7rX3HSq7tKOWmL8FdSs2AFx_h4o2YLWEEEpU/edit
2. In your project folder, open `docs/1MHR_All_Teams_Expanded.csv` (or download it if needed)
3. In Google Sheets: **File → Import → Upload** → choose the updated CSV
4. Select "Replace spreadsheet" or "Replace current sheet" so the new addresses overwrite the old ones

### Option B: Copy-paste the Address column
1. Open the updated CSV in Excel or Numbers
2. Select **only column G (Address)**
3. Copy
4. In your Google Sheet, select column G starting at row 2
5. Paste
